# 📹 AI YouTube Video Analyzer

Analyze any YouTube video using AI. This app generates structured insights, timestamps, and summaries.

## 🚀 Features

* 📊 Video overview
* ⏱️ Smart timestamps
* 🧠 Content breakdown
* 🎯 Key learning points

---

## 🛠️ Tech Stack

* Streamlit
* Groq (LLM)
* Agno Agents
* YouTube Tools

---

## 📦 Installation

```bash
git clone https://github.com/ayshaxsa/YouTube-Video-Analyzer.git
cd your-repo
pip install -r requirements.txt
```
